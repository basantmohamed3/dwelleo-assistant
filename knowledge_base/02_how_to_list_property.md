# How to List a Property on Dwelleo

Creating a property listing on Dwelleo takes about 10–15 minutes. This guide walks through the full process from the agent dashboard.

## Before You Start

Make sure you have the following ready: property title deed or authorization letter, at least 5 high-quality photos (minimum 1920x1080 resolution), accurate property dimensions, and the asking price in AED or SAR. For listings in Saudi Arabia, your Ejar number is also required. For UAE listings, the Trakheesi permit number must be entered before the listing can go live.

## Step-by-Step

1. Log into your agent dashboard at app.dwelleo.com.
2. Click "Create Listing" in the top navigation bar.
3. Select the property type: Apartment, Villa, Townhouse, Land, Commercial, or Off-Plan.
4. Choose the listing purpose: For Sale or For Rent.
5. Fill in the location section. The map auto-completes based on the address you enter. Confirm the pin is in the correct position before saving.
6. Enter property details: bedrooms, bathrooms, built-up area, plot area (if applicable), and amenities.
7. Upload photos. The first photo becomes the cover image. You can drag to reorder.
8. Add a description. AI-assisted description generation is available — click the magic wand icon to generate a draft based on the entered details.
9. Set your price. If you have access to AI Price Prediction, click "Get Price Suggestion" to see the predicted market range.
10. Add regulatory information (Ejar or Trakheesi).
11. Click "Publish" to make the listing live, or "Save as Draft" to come back later.

## Listing Approval

All new listings go through an automated review (typically 5–15 minutes) followed by a manual check for high-value properties (above 5 million AED/SAR). You'll receive an email and in-app notification once the listing is approved or if changes are requested.

## Editing a Live Listing

You can edit any field of a live listing at any time. Major changes (price changes over 20%, photo replacement, location change) trigger a re-review. The listing remains visible during re-review unless flagged.
